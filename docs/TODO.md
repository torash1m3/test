## 🚨 Технический долг (Допущения MVP)

Эти моменты были намерено упрощены на этапе быстрого прототипирования:
1. **Синхронизация локальных и облачных сборок**: При входе в аккаунт, если в облаке **уже** есть данные, локальные сборки не отправятся в облако. Нужна модалка "Объединить сборки?".
2. **Аватарки по ссылке**: Вместо полноценного Cloud Storage пока вставляется просто URL адрес картинки.
3. **Бесконечные лайки на Форуме**: Лайки работают через `increment(1)` без проверки, кто лайкнул. Пользователь может кликать бесконечно.
4. **Комментарии в документе**: Комменты добавляются через `arrayUnion` прямо в документ поста. Лимит Firestore на 1 документ = 1мб. На 3000-5000 среднего размера комментариев места хватит, но глобально лучше перевести в подколлекцию.
5. **Валидация длины данных**: Размер поля `pcSpecs` или текста комментария никак не лимитирован на уровне Firebase Rules.
